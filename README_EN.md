Personal portfolio website built with **Next.js**, **TypeScript**, and **Tailwind CSS**.  
The site showcases my projects, certificates, and contact information in a clean, modern, and professional layout.

---

## Short description
This portfolio was created to present my skills as a **Software Development student**.  
The main focus is on clarity, accessibility, responsive design, and a polished dark-mode UI with subtle animations.

---

## Technologies used
- **Next.js (App Router)**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **shadcn/ui**

---

## Features
- About me section
- Projects with links to GitHub
- Certificates (e.g. LinkedIn credentials)
- Downloadable CV (PDF)
- Modern dark theme
- Accessible focus states (keyboard friendly)
- Fully responsive (mobile & desktop)
